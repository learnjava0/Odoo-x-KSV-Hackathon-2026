## How to add screenshots

1. Run both the backend and frontend locally
2. Log in with the demo accounts (Admin, Manager, Procurement, Vendor)
3. Take screenshots of each screen
4. Save them to this folder with the exact filenames listed above
5. The README files in the root, `Frontend/`, and `backend/` folders will automatically display them
